# Frontend Interim Project

A real-time chat application that enables multiple users to send and receive messages seamlessly in a user-friendly interface.

## Features

### User Authentication
- Sign up with unique usernames
- Secure login functionality
- Logout capability 

### Real-Time Chat
- Display list of online users
- Send and receive messages instantly
- Support for both group and private messaging 

### Data Persistence
- Chat messages stored in localStorage for persistence across sessions
- User data managed in JSON format for reliability 

### User Experience Enhancements
- Timestamps displayed for all messages
- User profile viewing capability
- Validation of user credentials at login
- Unique username enforcement 

### User Interface
- Clean and intuitive design
- Organized layout with separate sections for:
  - Online users list
  - Chat message display area
  - Message input field with send button



## Technologies Used
- **HTML**
- **CSS**
- **JavaScript**
